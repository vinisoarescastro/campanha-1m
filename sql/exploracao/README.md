# sql/exploracao

Consultas exploratórias e de referência técnica, contagem de registros, faixas de datas, tipos de parcelas, mapeamento de campos e verificações iniciais de volume.

## Arquivos

| Arquivo | Descrição |
|---|---|
| `01_base_elegivel_campanha.sql` | Base de vendas elegíveis com status financeiro e aptidão ao sorteio — consulta oficial da campanha atual |

## Convenção de Nomenclatura

Arquivos são prefixados com número sequencial para indicar a ordem lógica de execução:

```
01_<descricao>.sql
02_<descricao>.sql
```
